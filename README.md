# brutasse

Brutasse is a Python module targetting network mischief at scale.

## Installing

```sh
apt install zmap
python3 -m venv .venv
source .venv/bin/activate
pip install .
```

## Running the cli

```sh
brutasse --help
```

## Building the documentation

```sh
cd docs
make html
xdg-open _build/html/index.html
```
